About Sentiment Analyzer

Sentiment Analyzer is a sophisticated AI-powered application for analyzing sentiment and emotions in text. Built with modern technologies and production-ready architecture, it demonstrates full-stack development expertise and ML integration.

Technology Stack

Backend

  • Flask 3.x with Application Factory pattern
  • SQLAlchemy 2.x with PostgreSQL/SQLite
  • Flask-RESTX with Swagger documentation
  • Celery for background task processing
  • JWT authentication

ML/NLP

  • VADER for quick sentiment analysis
  • Hugging Face Transformers
  • DistilRoBERTa emotion detection
  • 7 emotion classes detection
  • Batch processing with GPU support

Frontend

  • Tailwind CSS 3.x
  • Chart.js for visualizations
  • Server-Sent Events (SSE)
  • Responsive design

DevOps

  • Jenkins CI/CD pipeline
  • Comprehensive test suite
  • Code quality checks
  • Security scanning

Features

Real-time Analysis

Instant sentiment and emotion detection with sub-second response times.

Batch Processing

Upload CSV files with thousands of texts and track progress in real-time.

REST API

Full-featured API with Swagger documentation for integration.

7 Emotions

Detect anger, disgust, fear, joy, neutral, sadness, and surprise.
